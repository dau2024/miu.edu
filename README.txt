How to use:
1. Open index.html in a browser.

3. Replace contact details, fees, program details, and accreditation wording with official approved text before publishing.
4. The application form is a front-end demo. A developer can connect it to email, WhatsApp, PHP/MySQL, Firebase, or Google Sheets.
